# Project 004: 工业生产与销售 ERP 管理系统 (Web 端)

本项目是一个功能齐全、界面美观且业务逻辑高度还原的工业 ERP 管理系统前端演示项目。该系统通过 LocalStorage 实现全部数据的本地持久化，并包含采购、库存、BOM 生产及 FIFO 销售出库等核心工业 ERP 联动业务逻辑。

## 项目特点

1.  **全局单页应用 (SPA) 架构**：采用左侧导航 + 顶部状态栏 + 右侧动态内容区的经典 ERP 布局，提供极速响应的页面切换体验。
2.  **Tailwind CSS 现代化设计**：使用 Tailwind CSS 扁平化极简设计，结合柔和的 HSL 配色和微交互动画，呈现高端工业系统的质感。
3.  **安全库存预警**：自动监控库存，低于预水位时自动高亮报警。
4.  **BOM 生产联动与缺口计算**：支持 BOM 配置，下发工单前自动评估原料库存缺口，若原料不足则拦截生产。
5.  **先进先出 (FIFO) 扣减**：成品销售出库默认遵循 FIFO 原则扣减早期批次，并精确计算销售订单的毛利与成本。
6.  **双向批次追溯**：支持成品库存批次的下钻查看，能够清晰溯源该成品批次使用了哪些原料批次及对应的进价。

## 运行方式

无需任何后端依赖，双击或通过本地 Web 服务器（如 Live Server）打开 `index.html` 即可。

## 数据重置

如需重置为系统默认配置的工业基础数据（包含原料、成品、BOM配比和往来单位），可在系统工作台点击“重置系统数据”按钮。
